API

Kendi /v1/generate uç noktası, API anahtarları ve CDN (object storage) ayarları.

Kullanım

Görsel üretimi /v1/generate üzerinden SSE akışı ile yapılır. OpenAI uyumlu uç noktalar kaldırılmıştır.

        
event: status
data: {"stage":"generating","request_id":"gen-..."}

event: result
data: {"stage":"done","images":[{"png_url":"...","jpg_url":"...","url":"..."}]}

        
Alanlar: mode (image/video/audio) · aspect_ratio (1:1 / 16:9 / 9:16) · output_format (png/jpg) · gem_id (boşsa varsayılan gem) · store_media.
