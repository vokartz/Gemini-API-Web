API & Sistem Ayarları

API anahtarlarını yönet, CDN (nesne depolama) ayarla ve /v1/generate kullanımını gör.

API Anahtarları

Harici çağrılar Authorization: Bearer <API_KEY> ile yapılır. Anahtar yalnızca üretildiğinde bir kez açık gösterilir.

Nesne Depolama (CDN)

-

/v1/generate — Görsel Üretimi (SSE)

Temel URL
İstek
Akış olayları (text/event-stream) 
event: status
data: {"stage":"generating","request_id":"gen-..."}

event: result
data: {"stage":"done","images":[{"png_url":"...","jpg_url":"...","url":"..."}]}

event: error
data: {"stage":"error","detail":"...","status":502}
Alanlar 
prompt        → metin (zorunlu)
model         → gemini | gemini-3.1-pro | gemini-3.5-flash | gemini-3.1-flash-lite
mode          → image (varsayılan)
aspect_ratio  → 1:1 | 16:9 | 9:16
output_format → png | jpg (ikisi de CDN'e yüklenir)
gem_id        → özel gem id'si (boşsa varsayılan gem otomatik)
file_ids      → düzenleme için referans görsel id'leri (/v1/gemini/files)
store_media   → true ise CDN'e yüklenir
Diğer uç noktalar 
POST /v1/gemini/files      → dosya yükleme (file_id döner)
GET  /v1/custom-gems       → özel gem listesi
GET  /v1/gemini/media      → üretilen medya geçmişi
GET  /v1/status            → hesap havuzu ve kota durumu
