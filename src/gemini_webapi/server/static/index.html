OpenAI Uyumlu API Dokümanı

Bu sunucu, OpenAI uyumlu uç noktalar sunar. Mevcut OpenAI SDK'larını yalnızca base_url ve api_key değerlerini değiştirerek kullanabilirsiniz. API Key oluşturmak için Sistem Ayarları sekmesini kullanın.
Modeller 
gemini                 → varsayılan (gemini-3.1-pro takma adı)
gemini-3.1-pro
gemini-3.5-flash
gemini-3.1-flash-lite
GET /v1/models — Model listesi 
curl __BASE__/v1/models \
  -H "Authorization: Bearer $API_KEY"
POST /v1/chat/completions — Sohbet (Chat Completions) 
curl __BASE__/v1/chat/completions \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini",
    "messages": [{"role": "user", "content": "Merhaba!"}]
  }'
POST /v1/chat/completions — Akış (stream: true) 
curl -N __BASE__/v1/chat/completions \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini",
    "messages": [{"role": "user", "content": "Bir hikaye anlat"}],
    "stream": true
  }'
Python (openai SDK) 
from openai import OpenAI

client = OpenAI(
    base_url="__BASE__/v1",
    api_key="$API_KEY",
)

resp = client.chat.completions.create(
    model="gemini",
    messages=[{"role": "user", "content": "Merhaba!"}],
)
print(resp.choices[0].message.content)
POST /v1/images/generations — Görsel üretimi 
curl __BASE__/v1/images/generations \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini",
    "prompt": "Gün batımında bir dağ manzarası",
    "n": 1
  }'
POST /v1/responses — Responses API 
curl __BASE__/v1/responses \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini",
    "input": "Merhaba!"
  }'
POST /v1/files — Dosya yükleme (çok parçalı / multipart) 
curl __BASE__/v1/files \
  -H "Authorization: Bearer $API_KEY" \
  -F "file=@belge.pdf"
Not: Yukarıdaki örneklerde $API_KEY yerine kendi anahtarınızı, __BASE__ yerine ise yukarıdaki Temel URL'yi yazın (kod blokları zaten mevcut sunucu adresiyle otomatik doldurulur).

Manuel Cookie Ekle

En azından __Secure-1PSID alanını doldurun; ayrıca tam cookies JSON nesnesini veya tarayıcıdan dışa aktarılan diziyi de yapıştırabilirsiniz.
